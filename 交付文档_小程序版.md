## 技术方案

| Web 版 | 小程序版 |
|--------|---------|
| DOM/BOM API | WXML 数据绑定 + setData |
| IndexedDB | wx.setStorageSync |
| Chart.js CDN | 纯 Canvas 2D 自绘柱状图 |
| CSS light-dark() | @media prefers-color-scheme |
| HTML 模态弹窗 | TDesign Popup（底部滑出） |
| window.confirm | wx.showModal |
| Toast 自定义 | wx.showToast |
| 文件下载 | wx.setClipboardData（复制CSV） |
| 文件上传 | wx.chooseMessageFile |
| 原生 date input | picker mode="date" |

## 使用方式

1. 微信开发者工具打开 `miniapp/` 目录
2. `npm install` 安装 TDesign 依赖
3. 开发者工具 → 工具 → 构建 npm
4. 填写 `project.config.json` 中的 `appid`
5. 编译预览或上传审核
